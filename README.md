# HevTaskIO

HevTaskIO is a library of I/O operations for HevTask.

## How to Build

**Linux**:
```bash
git clone git://github.com/heiher/hev-task-io
cd hev-task-io
make
```

**Android**:
```bash
mkdir hev-task-io
cd hev-task-io
git clone git://github.com/heiher/hev-task-io jni
ndk-build
```
